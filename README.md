# HVN Commands

| Command        | Description                                    |
|----------------|------------------------------------------------|
| `hvn init`     | Initialize a new `.vcr` repository             |
| `hvn add`      | Track the current directory's folders          |
| `hvn commit`   | Submit changes to the server                   |

.vcr.ignore(file)->for ignoring files and directories
